This package will be for APE decoding one day, but now its only for reading APEv2 tags. You can read a tag by read-tag passing a bitreader to it. It will return a list of tag items in format 
((KEY-1 VALUE-1 :UTF-8 utf-8-p :r/w r/w-p)
 (KEY-2 VALUE-2 :UTF-8 utf-8-p :r/w r/w-p))
:utf-8-p is t when value is a text, not a binary, r/w-p is t when readonly flag is not set on that item.
